## Packages

- [adapter](adapter/README.md) — Package adapter provides WASM adapter runtime and loading functionality.
- [agent](agent/README.md) — Package agent implements HSM-based agent lifecycle and presence management.
- [config](config/README.md) — Package config provides configuration management with TOML format support.
- [conformance](conformance/README.md) — Package conformance provides the conformance test harness.
- [events](events/README.md) — Package events provides event dispatch functionality.
- [git](git/README.md) — Package git provides git operations for worktree management.
- [ids](ids/README.md) — Package ids provides identifier and normalization utilities for the amux system.
- [inference](inference/README.md) — Package inference provides local inference engine (liquidgen) functionality.
- [merge](merge/README.md) — Package merge provides git merge strategy implementation.
- [monitor](monitor/README.md) — Package monitor provides PTY observation and timeout detection.
- [paths](paths/README.md) — Package paths provides shared path resolution functionality.
- [pattern](pattern/README.md) — Package pattern provides pattern matching and action interfaces.
- [process](process/README.md) — Package process provides generic child process tracking functionality.
- [protocol](protocol/README.md) — Package protocol provides remote communication transport functionality.
- [pty](pty/README.md) — Package pty provides PTY session management for agents.
- [remote](remote/README.md) — Package remote implements remote host management, SSH bootstrap, NATS connectivity and control plane operations for distributed agent orchestration.
- [tui](tui/README.md) — Package tui provides terminal user interface functionality.
