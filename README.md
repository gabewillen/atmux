## Packages

- [cmd/amux](cmd/amux/README.md) — Package main implements the amux CLI client per spec §12.
- [cmd/amux-node](cmd/amux-node/README.md) — Package main implements the amux-node daemon per spec §12.
- [conformance/harness](conformance/harness/README.md) — Package harness provides the conformance testing harness per spec §4.3.1.
- [internal/adapter](internal/adapter/README.md) — Package adapter provides the WASM adapter runtime interface per spec §10.
- [internal/agent](internal/agent/README.md) — Package agent provides the agent actor model and HSM-driven lifecycle and presence state machines.
- [internal/config](internal/config/README.md) — Package config provides configuration management for amux per spec §4.2.8.
- [internal/errors](internal/errors/README.md) — Package errors provides error handling conventions and sentinel errors for amux.
- [internal/event](internal/event/README.md) — Package event provides event dispatch interfaces for amux per spec §9.
- [internal/inference](internal/inference/README.md) — Package inference provides the local inference engine interface for amux per spec §4.2.10.
- [internal/paths](internal/paths/README.md) — Package paths provides centralized filesystem path resolution for amux.
- [internal/snapshot](internal/snapshot/README.md) — Package snapshot implements the amux test snapshot functionality per spec §12.6.
- [internal/telemetry](internal/telemetry/README.md) — Package telemetry provides OpenTelemetry instrumentation for amux per spec §4.2.9.
- [pkg/api](pkg/api/README.md) — Package api provides public types for the amux system.
